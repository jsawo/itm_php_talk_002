Higher-order
functions

A higher order function is a function that takes
a function as an at leas one of its arguments,
or returns a function (or both)

Examples
in PHP

  • usort
  • array_filter
  • array_map
  • array_walk
$highOrderFunc = function($callback, $value) {
    return $callback($value) * 2;
};

$addFive = function($value) {
    return $value + 5;
};

$result = $highOrderFunc($addFive, 10);
