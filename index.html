Higher-order
functions

Funkcja wyższego rzędu (higher-order function)
to funkcja, która przyjmuje jako przynajmniej
jeden z parametrów inną funkcję, albo zwraca
funkcję (albo jedno i drugie)

Przykłady
w PHP

  • usort
  • array_filter
  • array_map
  • array_walk
$highOrderFunc = function($callback, $value) {
    return $callback($value) * 2;
};

$addFive = function($value) {
    return $value + 5;
};

$result = $highOrderFunc($addFive, 10);
